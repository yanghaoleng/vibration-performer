# Mickywa 个人网站设计沉淀

这份文档记录了本项目中的核心设计规范、色彩系统以及关键的交互动画，方便你在其他场景中延伸和复用。

## 1. 核心动画与交互 (Animations)

本项目十分注重微交互（Micro-interactions）的打磨：

- **文字入场律动 (Text Animation)**
  - 使用了开源的 [Pixelpoint Animate Text](https://pixelpoint.io/skills/animate-text/) 技术方案。
  - 在日程半弹窗中的预约意向文本切换时，使用了自下而上的平滑文字轮播替换（`BottomUpLettersSwap`），极大地提升了信息切换的灵动感。

- **弹簧缩放 (Spring Scale-in)**
  - 核心组件（如板块卡片）在初次渲染时，采用 `cubic-bezier(0.34, 1.56, 0.64, 1)` 的弹簧贝塞尔曲线，呈现带有轻微回弹的缩放出现（`animate-spring-scale-in`）。

- **按钮交互反馈 (Press Jump)**
  - “推荐列表”和“其他板块”内的操作按钮，点击瞬间会有向下的微缩放 (`scale: 0.95`)。
  - 释放后立刻弹起恢复，利用 `0.2s` 的延迟配合 `transform`，让每一次点击都有真实的按压阻尼感。

- **趣味贴纸生成 (Bubble-in)**
  - 新贴纸生成时，采用气泡式的弹簧放大动画：从 `0.3` 比例和下方透明状态，回弹放大到 `1.15` 再恢复到 `1.0`。
  - 拖拽贴纸时，贴纸放大至 `1.5` 倍并提升层级，同时带有实体感的白色细描边和真实的 CSS 阴影（`drop-shadow`），松手时平滑缩回，模拟真实的物理抓取体验。

## 2. 色彩系统 (Color Palette)

本项目为亮色/暗色模式（Light / Dark）做了精细的双套色彩适配，其中暗色模式并不仅仅是简单的反色，而是将主题色混合了半透明度，保证了视觉上的透气感。

### 全局背景色
- **Light**: `#F9F9F9`
- **Dark**: `#1E1E1E` (搭配柔和的毛玻璃蒙版)

### 日程主题色
- **主文本/边框**: 
  - Light: `#3A3A3A`
  - Dark: `#FFFFFF`
- **强调色/链接**: 
  - Light: `#083A8E` (深蓝)
  - Dark: `#D3F1FF` (浅蓝)

### 独立板块主题色

在“找我耍”系列板块中，每个子板块都有独立的色系：

1. **找我耍 (主日程区域)**
   - **Light**: 背景 `#D3F1FF`，文字/SVG `#083A8E`
   - **Dark**: 背景 `#083A8E/25`，文字/SVG `#D3F1FF` (浅蓝着色)

2. **刷抖音**
   - **Light**: 背景 `#FFE4E6`，文字 `#8E083A`
   - **Dark**: 背景 `#8E083A/25`，文字/SVG `#FFD3F1` (浅粉着色)

3. **一起 Vibe**
   - **Light**: 背景 `#FEF3C7`，文字 `#8E6A08`
   - **Dark**: 背景 `#8E6A08/25`，文字/SVG `#FFF4D3` (浅黄着色)

4. **随便聊**
   - **Light**: 背景 `#E6F4EA`，文字 `#088E3A`
   - **Dark**: 背景 `#088E3A/25`，文字/SVG `#D3FFD3` (浅绿着色)

## 3. 组件 UI 规范

- **大圆角 (Border Radius)**
  - 外层大卡片统一使用 `28px` 圆角。
  - 内层白底区域统一使用 `18px` 圆角。
  - 按钮及小元素统一使用 `12px` 圆角。
  
- **半弹窗 (Half Modal)**
  - 移动端：底部吸附，顶部使用 `24px` 圆角，取消底部圆角。
  - 桌面端：屏幕正中，四周使用 `24px` 圆角。
  - 背景遮罩层为 30% 透明度的纯黑/深灰，带有渐隐过渡。

你可以通过调整 `src/index.css` 和 `tailwind.config.js` 来进一步定制这些设计规范。
