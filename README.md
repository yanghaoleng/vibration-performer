# 震动演奏机

一个用于录制、编辑、播放和导出震动曲线的静态网页。它把触摸滑块或鼠标滚轮的演奏过程记录为带时间码的振幅曲线，并导出为 JSON。

## 功能

- 手机端使用滑块控制振幅，桌面端使用鼠标滚轮控制振幅
- 录制过程按 `1/3` 速度写入时间轴
- 停止录制后自动平滑曲线
- 支持拖动曲线点手动调整振幅
- 支持音频播放反馈；设备支持时会调用 `navigator.vibrate`
- 每个片段包含随机中文名、时间码、持续时间和曲线点位
- JSON 导出时会下载文件，并尝试复制到剪贴板
- 支持中文、英语、日语
- 支持亮色、暗色、跟随系统三种主题
- 支持常见音视频编辑式快捷键

## 快捷键

- `R`: 开始或停止录制
- `Space` / `K`: 播放或停止播放
- `S`: 平滑曲线
- `E` / `Cmd/Ctrl + E`: 导出 JSON
- `Cmd/Ctrl + Z`: 撤销曲线编辑
- `Cmd/Ctrl + Shift + Z`: 重做曲线编辑
- `Delete` / `Backspace`: 清空当前片段
- `Arrow Up` / `+`: 增加振幅
- `Arrow Down` / `-`: 减少振幅
- `0`: 振幅归零
- `Esc`: 停止当前动作并关闭菜单

## 本地预览

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173
```

## 浏览器说明

网站是纯静态实现，重点适配 Safari。iOS Safari 通常不开放 Web Vibration API，因此手机线性马达演示会在支持 `navigator.vibrate` 的浏览器或安卓设备上生效；不支持时仍可通过低频音效预览震动节奏。
